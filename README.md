# NN-vs-logisticRegression
Comparison of a NN and a Logistic Regression model to predict breast cancer using Sklearn and PyTorch.

Specs: A dataloader was used not for efficiency but as a good practice:

    * Batching
    * Shuffling (Not required but used)

    * num_workers > 0 for parallel data loading
